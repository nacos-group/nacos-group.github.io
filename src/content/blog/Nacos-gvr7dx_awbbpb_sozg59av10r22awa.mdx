“零信任架构” 是一种安全概念，它要求在任何时候不对任何请求默认信任，无论它的来源内部还是外部。服务安全性已成为企业的核心关切，我们将在**4月12日（周五）和4月13日（周六）在北京阿里中心**探讨如何设计和落地微服务的安全零信任架构。<br />欢迎点击报名：[https://survey.aliyun.com/apps/zhiliao/Nz7WNQOt0](https://survey.aliyun.com/apps/zhiliao/Nz7WNQOt0)<br />**此次议题我们会深入探讨：**

1. 网关安全，如何通过 Higress 网关做好入口流量的安全和监控；
2. 微服务调用安全，如果保障 Dubbo 与 Spring Cloud 服务间通信的完整性和机密性；
3. 配置与消息安全，防止 Nacos、RocketMQ 敏感信息的泄露及不当访问
4. 容器 Kubernetes 零信任安全监控
