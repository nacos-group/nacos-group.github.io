# Nacos Server 下载

:::tip

Nacos Server **2.3.0** 已正式发布。支持持久化服务通过gRPC协议注册和注销，修复问题若干。

欢迎[下载和试用](https://github.com/alibaba/nacos/releases/tag/2.3.0).

## 系统要求

:::note

以下是部署Nacos Server的最小系统要求。如果您的环境无法满足系统最小要求，可能会导致无法部署和启动Nacos Server。

:::

|             |                                                                                                                         |
|-------------|------------------------------------------------------------------------------------------------------------------------------------|
| **JDK/JRE** | 8及以上  |
| **CPU**  | 1核及以上，支持64位CPU    |
| **内存** | 2G及以上 |
| **硬盘** | 无最小要求，根据保留日志自行调整|
| **操作系统** | Linux, Mac OS X, Windows |
